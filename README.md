# Vending Machine Simulation
## by Kayla Reyes and Viktor Karpenko
### Data Structures and Algoritms with Dr. Todd Wolfe at SUNY Empire State University

## Project Overview
This project is a Java-based vending machine simulation designed to demonstrate core programming concepts related to data structures and algorithms. The program allows users to select items, process payments, and receive appropriate output based on their interaction.

The goal of this project is to apply fundamental programming concepts such as arrays, conditionals, loops, and input validation in a real-world scenario.

## Objectives
- Implement a functional vending machine simulation in Java
- Practice working with arrays and basic data structures
- Apply conditional logic and control flow
- Handle user input and validate transactions
- Demonstrate clean, structured, and maintainable code

## Core Features (MVP - Phase 1)
- Display a menu of items with prices
- Allow user to select an item
- Accept user payment input
- Check if sufficient funds are provided
- Dispense item and return change
- Handle invalid inputs gracefully

## Potential Enhancements (Future Scope)
- Inventory tracking (item quantities)
- Out-of-stock handling
- Multiple transactions in one session
- Admin mode for restocking items
- Sales tracking and reporting
- Improved user interface and experience
